# vanilla-chat-app


Here are the two primary environment variable names you need to set manually in your Vercel dashboard for your chat app to work:

    OPENROUTER_API_KEY: This stores your secret key from OpenRouter to connect to the AI models (like Qwen, Stepfun, or Nvidia).

    APP_PASSWORD: This stores the custom password you created to lock your frontend UI so only you can access it.
